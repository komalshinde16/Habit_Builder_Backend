Habit Builder is a backend REST API developed using Spring Boot that helps users manage their daily habits.  
It allows users to create, view, update, and delete habits in a simple and structured way.

---

## Tech Stack
- Java  
- Spring Boot  
- Spring MVC  
- Spring Data JPA  
- MySQL  
- Maven  

---

## Features
- Add a new habit  
- View all habits  
- Update habit details  
- Delete habit  

---

## API Endpoints
| Method | Endpoint | Description |
|------|---------|-------------|
| POST | /habits | Create a new habit |
| GET | /habits | Get all habits |
| PUT | /habits/{id} | Update a habit |
| DELETE | /habits/{id} | Delete a habit |

---

## Testing
All APIs can be tested using **Postman**.
